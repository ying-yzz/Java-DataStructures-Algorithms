# Práctica 2: Optimización de Algoritmos de Ordenación

Este proyecto se centra en el diseño y validación de una implementación eficiente del algoritmo Merge Sort.

### Características técnicas:
* **Merge Sort Eficiente**: Modificación del método de mezcla para reducir el movimiento de datos y optimizar el uso de memoria.
* **Algoritmia Comparativa**: Análisis de eficiencia entre algoritmos de ordenación directa $O(n^2)$ y algoritmos rápidos $O(n \log n)$.
* **Pointer Sorting**: Aprovechamiento del bajo coste de movimiento de referencias en Java para mejorar el rendimiento del algoritmo.
* **Genericidad y Estabilidad**: Implementación de métodos genéricos que preservan el orden inicial de datos idénticos.
