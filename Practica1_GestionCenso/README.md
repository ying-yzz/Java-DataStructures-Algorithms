# Práctica 1: Gestión de Censo de Población

Este proyecto implementa un registro de habitantes para administraciones públicas.

### Características técnicas:
* **Estructura de Datos:** Uso de una lista enlazada ordenada (`LEGListaConPIOrdenada`).
* **Algoritmia:** Inserción ordenada ascendente basada en la interfaz `Comparable`.
* **Lógica de Negocio:** Gestión de objetos `Habitante` con criterios de comparación por apellidos y DNI.
